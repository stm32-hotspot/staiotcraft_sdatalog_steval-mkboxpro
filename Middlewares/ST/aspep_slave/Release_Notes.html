Purpose

The ASPEP slave middleware implements the ASPEP (Asymmetric Serial Packet Exchange) slave protocol layer to provide payload encapsulation with the ASPEP header. See readme for further details.

For a complete reference of the ASPEP protocol, see MCSDK documentation. Official documentation on ASPEP protocol can be found in X-CUBE-MCSDK installation folder also.
