# Purpose

The **SimpleSerialTL protocol layer** implements the SSTL protocol layer to provide the application with a set of Sync/Async API over the ASPEP layer.
See [readme](.\readme.html) for further details.

This simple point-to-point protocol allows to transfer packets on a serial link, like a UART peripheral. 
It's a master/slave protocol that supports both synchronous and asynchronous packets, and it's tailored specifically to multi-channel data streaming applications.
